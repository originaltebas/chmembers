Este es una pequeña aplicación en Python Flask 
para la gestión de miembros de una iglesia.

El foco está en la simplicidad: tiene que ser simple de entender,
fácil de usar y con las funcionalidades justas.

Agregar, modificar y quitar miembros. Definir familias y personas que viven
en la misma casa. Asignacion de grupos caseros (o pequeños o células)
